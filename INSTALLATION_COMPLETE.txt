================================================================================
  TROUBLESHOOTING
================================================================================

PROBLEM: "Python not found"
SOLUTION: Install Python 3.8+ from python.org

PROBLEM: "Module not found"
SOLUTION: pip install -r requirements.txt

PROBLEM: "Pack not found"
SOLUTION: Make sure pack folder is in input/ with pack.mcmeta

PROBLEM: "Permission denied"
SOLUTION: chmod +x main.py OR run with: python3 main.py

PROBLEM: Resource pack doesn't appear in Minecraft
SOLUTION: Check resource packs folder path, verify manifest.json valid

FOR MORE HELP:
- See README.md for comprehensive troubleshooting
- Check logs/ folder for detailed conversion logs
- Run with -v flag for verbose output: python3 main.py pack -v

================================================================================
  PERFORMANCE TIPS
================================================================================

For Large Resource Packs:

• Monitor system resources: free -h, df -h
• Run during non-peak hours
• Close other applications
• Use SSD for faster I/O
• For Termux: termux-wake-lock to keep CPU awake
• Monitor progress: tail -f logs/*.log

Expected Times:
- Small pack (< 50MB):  1-2 minutes
- Medium pack (50-200MB): 3-5 minutes
- Large pack (> 200MB): 10+ minutes
