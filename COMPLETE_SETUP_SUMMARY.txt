QUICK START (Choose Your Platform)
================================================================================

WINDOWS/macOS/LINUX:
  1. Install Python 3.8+ from python.org
  2. Install Git from git-scm.com
  3. git clone https://github.com/PickoCore/GVerterx.git
  4. cd GVerterx
  5. pip install -r requirements.txt
  6. python main.py
  
ANDROID (Termux):
  1. Install Termux from F-Droid
  2. pkg update && pkg upgrade -y
  3. pkg install python git ffmpeg build-essential -y
  4. termux-setup-storage
  5. cd ~/storage/downloads
  6. git clone https://github.com/PickoCore/GVerterx.git
  7. cd GVerterx
  8. pip install -r requirements.txt
  9. python main.py

HOW TO USE (3 SIMPLE STEPS)

TROUBLESHOOTING QUICK REFERENCE
================================================================================

Problem                              Solution
─────────────────────────────────────────────────────────────────────────────
Python not found                     Install from python.org, check PATH
Git not found                        Install from git-scm.com
ModuleNotFoundError                  pip install --force-reinstall -r requirements.txt
FFmpeg not found                     Optional - install for better audio quality
Permission denied                    chmod +x main.py (Linux/macOS/Termux)
Converter won't start                Check Python 3.8+, verify dependencies
Conversion too slow                  Reduce TEXTURE_SCALE in config.py
Out of memory                        Lower TEXTURE_SCALE to 0.25 or 0.5
Storage full                         Clean temp/, delete old output/
Output not created                   Check logs/converter.log for errors
Termux storage denied                Run: termux-setup-storage

See TROUBLESHOOTING.md for detailed solutions!
